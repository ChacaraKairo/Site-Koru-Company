# Koru Company

Site institucional estático da Koru Company, refeito com HTML, CSS e JavaScript puro.

## Estrutura

- `index.html`: página inicial
- `servicos.html`: serviços da empresa
- `produtos.html`: página preparada para adicionar produtos, logos e explicações
- `projetos.html`: exemplos de soluções
- `sobre.html`: apresentação da empresa e fundador
- `contato.html`: formulário que abre o WhatsApp com a mensagem organizada
- `politica-de-privacidade.html`: política de privacidade
- `assets/css/styles.css`: estilos globais
- `assets/js/main.js`: menu mobile, links de WhatsApp e formulário
- `public/imagens`: imagens e logos usadas no site

## Como rodar

Você pode abrir o `index.html` diretamente no navegador ou iniciar um servidor local:

```bash
npm run dev
```

Depois acesse:

```text
http://localhost:3000
```

## Editando produtos

Na página `produtos.html`, substitua os blocos com `Nome do Produto` e `Logo` pelos nomes, imagens e descrições reais dos produtos da Koru.
